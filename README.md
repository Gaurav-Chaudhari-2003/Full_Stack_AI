## üìÇ Project Structure

The repository is organized into weekly modules, each focusing on a specific pillar of Full Stack AI development:

*   **`week1`** - Foundations and Environment Setup.
*   **`week2`** - Data Engineering & Exploration.
*   **`week3`** - Model Selection & Training.
*   **`week5_week6`** - Advanced Implementation & Integration.
*   **`pdf_to_img.ipynb`** - Utility tool for converting PDF documents into images for OCR or Vision-based tasks.
